# Stock AI V1.4.2

## 你可以直接上传到 GitHub Pages

V1.4.2 是自动行情架构版，不需要先上传 V1.4.0、V1.4.1。

## 更新内容

- 自动获取数据按钮
- 支持自定义 API 地址
- 真实 API 失败时，自动切换内置备用数据
- 没有备用数据时，允许手动填写继续分析
- 保留 V1.3.2 的买点等级、仓位参考、止损参考、核心风险
- 保留未收录股票可分析逻辑

## 重要说明

如果不配置自己的后端 API，页面不能保证实时获取股票行情。
GitHub Pages 是静态网站，适合放前端；真正稳定的行情获取需要后端接口。
