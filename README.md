# Stock AI V1.5 Ultimate Pro Sprint 3

V1.5 Ultimate Pro 是 Stock AI 的分析大脑版本，重点不是联网，而是把职业交易员式分析流程做成熟。

## 本次升级：Sprint 3 风险系统重构

- 保留 Sprint 1 的大位置五档系统。
- 保留 Sprint 1 的小位置五档系统。
- 保留“大位置优先原则”。
- 保留 Sprint 2 的位置评分、趋势评分、量能评分、结构评分。
- 保留 Sprint 2 的 40 分制总评分和买入环境评级。
- 新增风险评分系统：位置风险、趋势风险、量能风险、压力风险。
- 新增总风险 20 分制。
- 新增 A/B/C/D/E 综合风险等级。
- 报告新增【风险评分】模块。
- Summary Cards 新增“总风险”。

## 说明

V1.5 Ultimate Pro 仍然是不联网版本，专注于把分析逻辑打磨成熟。真实行情、自动获取 MA20/MA60、历史高低点和成交量留给 V2.0。

本项目仅用于学习和辅助分析，不构成投资建议。
