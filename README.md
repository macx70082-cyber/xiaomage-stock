# Stock AI V1.5 Pro

## 核心升级

V1.5 Pro 从“评分工具”升级为“老手决策版”。

## 重要变化

- 删除大量内置股票数据库
- 不再靠人工记录股票是否存在
- 前端只做 A股代码格式判断
- 未来联网后，由 API 查询全市场股票数据
- 手动填写仍作为备用方案

## 新增功能

- 老手决策树
  - 大位置
  - 趋势结构
  - MA20距离
  - 量能
  - 风险
  - 交易计划

- 大位置细分
  - 历史底部区
  - 低位区
  - 中位区
  - 高位区
  - 历史顶部区

- 支持理由与反对理由
- 支撑位与压力位
- 止损参考
- 第一目标、第二目标
- 失败信号
- 老手口吻结论
- 参考仓位

## 后续路线

V1.6 Online：
接入真实股票数据接口，输入代码后自动获取股票名称、价格、MA5、MA10、MA20、MA60、历史高低点。
