# Stock AI V1.5 Ultimate

V1.5 Ultimate 是 Stock AI 的分析大脑版本，重点不是联网，而是把职业交易员式分析流程做成熟。

## 核心升级

- 大位置 / 小位置智能判断
- 多空评分系统
- 支撑压力自动计算
- 风险拆解：位置、趋势、量能、压力
- 交易计划：低吸观察、突破观察、止损、放弃条件
- 案例匹配系统
- 常见错误提醒
- 职业交易员报告格式

## 文件结构
